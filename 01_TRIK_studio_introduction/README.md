# TRIK Studio introduction
TRIK robotics set has an IDE with embeded simulator. There are visual and text programming avaliable from the shelf (Java Script and Python without advanced libraries, like Math.js or Numpy).
The first step will be development of the simple code with test in the simulation and real robot. Use the functions play sound, led blinking or display values, console printing, data logging. Here is a nice instruction for the robot testing https://roboticsknowledgebase.com/wiki/robotics-project-guide/test-and-debug/
Use the TRIK help for the latest updates https://help.trikset.com/en
