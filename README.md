# Robotics-course
This repository contains the Robotic course materials for the introduction to the discipline. It's covering the basic concepts of the PID control, system identification, modeling, localization and trajectory control.\
The code for the lab works developed in several realizations can be used in TRIK Studio Software and ROS2. Basic hardware set is differential drive TRIK robot with Rasberry Pi RPi connected via USB to RJ45 Network Adaptor with ASIX chipset.\
The code for the TRIK robot can be launched with TRIK only, but ROS2 needs RPi computational power and the TRIK script launched to transmit the data to ROS2.
